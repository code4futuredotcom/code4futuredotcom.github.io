Leo Editor 簡介

Code4future | Tue 07 November 2017

Code4future 的網誌是利用 Leo Editor 編輯管理, 位於 content 目錄中的 Markdown 檔案是網誌的主要內容, 以 Leo Editor 的節點管理每一則 .md 檔案, 然後再將 Pelican 轉檔用的指令存成 Leo Editor 的 button, 在正常情況下, 只要利用滑鼠點到該 button 所在的路徑後, 再點擊命令執行轉檔.

之後再利用 Leo Editor WWW 按鈕伺服檢查所轉換的網誌. 由於上述各種操作都在 Leo Editor 中執行, 因此本網誌將簡單介紹相關功能.
