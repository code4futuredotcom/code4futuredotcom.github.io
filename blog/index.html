Pelican 靜態網頁產生器

Code4future | Tue 07 November 2017

Pelican 是一種靜態網頁產生程式, 以 Python3 編寫, 可以將作者所編輯的 reStructuredText 或 Markdown 檔案, 配合網站 Template 及設定檔案, 轉換為一組網誌 html 超文件, 只需要 WWW 伺服器就可以在網站上呈現, 目前的 Github Pages 服務應該最適合靜態網誌的寄存.
