網誌可利用 Slug 自訂轉檔後的 html 名稱

User | Tue 07 November 2017

以英文命名的網誌, 可以無需使用 slug 設定轉檔後的檔案, 因為 Pelican 會自動採英文 title 名稱作為 html 檔名.

但是以中文命名的網誌, 則建議使用 Slug 欄位自行決定有意義的 html 檔案名稱, 假如沒有 Slug 欄位設定, Pelican 則會以拼音的方式將中文網誌標題, 轉為英文拼音名稱.html
